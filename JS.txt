Rangkuman Javascript

Definisi : 
- javascript merupakan bahasa pemrograman tingkat tinggi

History :
- javascript pertama kali diciptakan oleh brendan eich seorang programmer netscape pada tahun 1995

- javascript awalnya bernama mocha kemudian berubah menjadi livescript dikarenakan bahasa pemrograman java sedang populer maka livescript pun diubah menjadi javascript

- javascript diadopsi diluar netscape kemudian distandarisasi oleh ECMA dan menjadi ECMAscript, pada tahun 2000-2010 ECMAscript3 sedang populer dan antara tahun tersebut ECMAscript4 juga dikembangkan dan ditutup pada tahun 2008 karena alasan tertentu. pada tahun 2009 ECMAscript5 rilis hanya dengan sedikit perbaikan. pada tahun 2015 ECMAscript6 rilis dengan banyak perubahan termasuk ide pada ECMAscript4

sedikit penjelasan : 
- expression yaitu kode yang mehasilkan nilai

- statement yaitu kode yang dapat melakukan aksi

- identifier yaitu nama variabel

- setiap nilai pada javascript mewarisi sifat boolean yaitu dengan menggunakan truthy dan falsy
	- contoh falsy yaitu number 0, bigInt 0n, string kosong seperti "", null, undefine, NaN
