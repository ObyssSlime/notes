Rangkuman Javascript

Definisi : 
- javascript merupakan bahasa pemrograman tingkat tinggi

History :
- javascript pertama kali diciptakan oleh brendan eich seorang programmer netscape pada tahun 1995

- javascript awalnya bernama mocha kemudian berubah menjadi livescript dikarenakan bahasa pemrograman java sedang populer maka livescript pun diubah menjadi javascript

- javascript diadopsi diluar netscape kemudian distandarisasi oleh ECMA dan menjadi ECMAscript, pada tahun 2000-2010 ECMAscript3 sedang populer dan antara tahun tersebut ECMAscript4 juga dikembangkan dan ditutup pada tahun 2008 karena alasan tertentu. pada tahun 2009 ECMAscript5 rilis hanya dengan sedikit perbaikan. pada tahun 2015 ECMAscript6 rilis dengan banyak perubahan termasuk ide pada ECMAscript4

sedikit penjelasan : 
- expression yaitu kode yang mehasilkan nilai

- statement yaitu kode yang dapat melakukan aksi

- identifier yaitu nama variabel

- setiap nilai pada javascript mewarisi sifat boolean yaitu dengan menggunakan truthy dan falsy
	- contoh falsy yaitu number 0, bigInt 0n, string kosong seperti "", null, undefine, NaN

- penggunaan object
	- terdapat metode delete, code: delete.object.key
	
	- bila kay menggunakan spasi maka untuk mengaksesnya, code: object["space key"]

	- terdapat destruction object
		- membuat key object menjadi variabel lokal, code: object{key:value}<enter>{key}=object
		
		- merename key dengan menambahkan nama key baru, code: {key:renameKey}=object

- penggunaan array
	- terdapat metode push untuk menambahkan data, code: object.push("data")

	- terdapat metode pop untuk menghapus data dari belakang, code: object.pop()

	- terdapat metode shift untuk menghapus data dari depan, code: object.shift()

	- terdapat metode unshiift untuk menambahkan data dari depan, code: object.unshift("data")

	- terdapat metode delete untuk menghapus data dengan indeks tapi hasilnya memiliki spasi atau null, code: delete.object[indeks]

	- terdapat metode splice untuk menghapus data dengan indeks sampai bersih data, code: object.splice(2,1) artinya pada indeks 2 sebanyak 1 elemen

	- menggunakan spread operator untuk menggabungkan indeks array, code: ...array

	- terdapat destruction array
		- penggunaannya sama dengan object hanya diganti {} dengan []

- penggunaan map
	- penggunaannya sama seperti object tetapi propertynya bisa menggunakan semua tipe data 	
	- terdapat metode get()

	- terdapat metode set()

	- penggunaannya, code: map=new Map([key,value])
